# Шаг 4: Настройка Firewall

## Настройка firewall

[X] Открыть взаимодействие для сети WireGuard в firewall

**Через web-интерфейс:**

- Перейти в "Сетевые правила" → "Межсетевой экран", выбрать созданное подключение `wg0`
- Создать правило:
  - Действие: `Разрешить`
  - Протокол: `IP`
  - IP-адрес источника и назначения: `Любой`
- Сохранить
