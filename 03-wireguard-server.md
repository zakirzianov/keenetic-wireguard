# Шаг 3: Настройка WireGuard сервера

## Настройка `WG-LAN`

[x] Установить VPN-сервер WireGuard

**Через web-интерфейс:**

- Перейти в "Управление" → "Параметры системы" → "Общие настройки системы" → "Изменить набор компонентов"
- Найти и установить компонент "WireGuard VPN"
- Дождаться завершения установки
- Роутер может потребовать перезагрузку
- После этого раздел `WireGuard` появится на странице "Другие подключения"

[x] Выбрать подсеть для WG-LAN (например, `10.0.0.0/24`) и определить IP-адреса сервера и клиентов

**Планирование адресации:**

```text
WG-LAN подсеть: 10.0.0.0/24
Сервер (Keenetic): 10.0.0.1
Клиент 1 (PC): 10.0.0.2
Клиент 2 (OpenWRT): 10.0.0.3
... и так далее для других клиентов
```

**Запишите выбранную схему адресации для использования в дальнейших настройках!**

[x] Выбрать UDP-порт для WireGuard (стандартный `51820` или другой)

**Рекомендации:**

- Стандартный порт: `51820`
- Можно использовать нестандартный порт для дополнительной обфускации (например, `51821`, `12345`)
- Запишите выбранный порт для дальнейшей настройки firewall

**Выбранный порт WireGuard:** (например, `51820`)

[x] Сгенерировать ключи для сервера и клиента

**⚠️ ВАЖНО:** Сохраните все ключи в безопасном месте! Приватные ключи не должны быть доступны посторонним.

**Запишите ключи:**

```text
Server Private Key: [сохранить]
Server Public Key: [сохранить]
Client1 Private Key: [сохранить]
Client1 Public Key: [сохранить]
```

[x] Настроить конфигурацию сервера WG (указать подсеть, порт, ключи)

**Через web-интерфейс:**

- Перейти в раздел "Интернет" → "Другие подключения" → "WireGuard"
- Нажать "Добавить подключение"
- Указать параметры:
  - Название: `WG-Server` или `wg0`
  - Приватный ключ сервера: (вставить из `server_privatekey`)
  - Адрес интерфейса: `10.0.0.1/24`
  - Порт прослушивания: `51820` (или выбранный вами)
- Сохранить

[x] Добавить клиента в конфигурацию сервера WG

**Через web-интерфейс:**

- В разделе "WireGuard" выбрать созданное подключение `wg0`
- Нажать "Добавить подключение"
- Сгенерировать пару ключей
- Указать IPv4 адрес в формате `10.0.0.1/24`
- Указать порт для подключения
