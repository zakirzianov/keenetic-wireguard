# Шаг 1: Настройка роутера Keenetic

## Настройка роутера `Keenetic`

### Резервное копирование

[x] Создать резервную копию настроек роутера

**Через web-интерфейс:**

- Открыть web-интерфейс Keenetic (обычно `http://192.168.1.1` или `http://my.keenetic.net`)
- Перейти в раздел "Управление" → "Параметры системы" → "Системные файлы"
- Нажать кнопку "Сохранить на компьютер" для создания резервной копии файла `startup-config`
- Сохранить файл `.cfg` на локальный компьютер с указанием даты (например, `KN-1010_stable_4.03.C.6.3-9_router_startup-config.txt`)

### Добавление пользователя

[x] Добавить пользователя в роутер, назначить ему сложный пароль

**Через web-интерфейс:**

- Перейти в раздел "Управление" → "Пользователи и доступ"
- Нажать "Добавить пользователь"
- Указать имя пользователя (например, `myuser`)
- Указать сложный пароль (рекомендуется использовать генератор паролей, минимум 16 символов)
- Назначить все права доступа
- Сохранить изменения
- Убедиться, что пользователь `myuser` появился в перечне пользователей с отметкой `Да` в поле "Права администратора"
- Выйти из системы ("Шестеренка" в правом верхнем углу интерфейса, `Выйти`)
- Войти в систему под новым пользователем `myuser`

[x] Запретить пользователю `Admin` доступ к роутеру

**⚠️ ВАЖНО:** Сначала убедитесь, что можете войти под новым пользователем, и только потом отключайте `Admin`!

**Через web-интерфейс:**

- Перейти в раздел "Управление" → "Пользователи и доступ"
- Выбрать пользователя `Admin`
- Снять все галочки доступа (только после проверки доступа под новым пользователем!)
