# Настройка локальной и WG-сети на базе роутера Keenetic

## Описание

Пошаговое руководство по настройке безопасной домашней сети на базе роутера Keenetic с VPN-сервером WireGuard и VPN-сервером Wireguard (с обфускацией параметрами AWG 2.0).

### Архитектура сети

Настройка создаёт две сетевые зоны:

1. **Keenetic-LAN** (например, 192.168.1.0/24) - локальная сеть за роутером Keenetic
2. **WG-LAN** (например, 10.0.0.0/24) - сеть WireGuard VPN для безопасного удалённого доступа
3. **AWG-LAN** (например, 10.1.1.0/24) - сеть WireGuard VPN ( с обфускацией параметрами AWG 2.0)  для безопасного удалённого доступа
   

### Ключевые принципы безопасности

- Web-интерфейс роутера доступен только из Keenetic-LAN, WG-LAN и AWG-LAN (не напрямую из интернета)
- SSH-доступ ограничен локальными сетями и WG-сетью
- Локальные сервисы доступны через WG-туннель, но не напрямую из интернета

## Важные замечания

### Безопасность

- Всегда создавайте резервные копии перед внесением изменений
- Не закрывайте текущую SSH-сессию, пока не убедитесь, что новая работает

### Сеть

- Убедитесь, что выбранные подсети не пересекаются
- Запишите все выбранные IP-адреса и порты
