# Шаг 2: Настройка SSH

## Настройка доступа через `ssh`

[x] Установить `ssh` сервер

**Через web-интерфейс:**

- Перейти в раздел "Управление" → "Параметры системы"
- Найти секцию "Обновление и компоненты KeeneticOS" и нажать кнопку → "Изменить набор компонентов"
- Найти компонент "Сервер SSH" и включить галочку "Установлен"
- Порт по умолчанию: `22` (позже изменим на нестандартный)

[x] Добавить пользователя в перечень имеющих доступ через `ssh`

**Через web-интерфейс:**

- Перейти в раздел "Управление" → "Пользователи и доступ"
- Выбрать созданного пользователя (`myuser`)
- Включить галочку "Командная строка"

[x] Изменить стандартный порт `ssh` на нестандартный

**Через SSH:**

```bash
# Подключиться к роутеру
ssh myuser@192.168.1.1

# Отредактировать конфигурацию SSH
(config-ssh)> port 2022
Ssh::Manager: Port changed to 2022.

# Сохранить конфигурацию
(config)> system configuration save
Core::ConfigurationSaver: Saving configuration...
```

**После смены порта подключаться нужно так:**

```bash
ssh -p 2022 myuser@192.168.1.1
```
