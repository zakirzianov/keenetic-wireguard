## Обзор проекта

Это проект документации и конфигурации для настройки безопасной домашней сети на базе роутера Keenetic с VPN-сервером WireGuard. Репозиторий содержит пошаговые инструкции для настройки:

- Роутера Keenetic с доступом по SSH и усилением безопасности
- VPN-сервера WireGuard на Keenetic (WG-LAN)
- Клиентов WireGuard на PC и роутерах OpenWRT
- Локальной DNS-конфигурации для пользовательских доменных имён
- Обратного прокси с SSL-сертификатами
- Правил firewall для безопасного удалённого доступа

## Архитектура сети

Настройка создаёт три сетевые зоны:

1. **Keenetic-LAN** (например, 192.168.1.0/24) - локальная сеть за роутером Keenetic
2. **WG-LAN** - сеть WireGuard VPN для безопасного удалённого доступа
3. **OpenWRT-LAN** (например, 192.168.99.0/24) - вторичная локальная сеть с роутером OpenWRT в качестве клиента WG

Ключевые принципы безопасности:
- Web-интерфейс роутера доступен только из Keenetic-LAN и WG-LAN (не напрямую из интернета)
- SSH-доступ ограничен локальными сетями и WG-сетью
- Локальные сервисы (например, `srvok.ckox`) доступны через WG-туннель, но не напрямую из интернета
- Пользовательский DNS на Keenetic разрешает локальные доменные имена (например, `*.srvok.ckox`)
